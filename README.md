# 🌾 AgriEdge Orange Ltd – Smart Order Management System

> Transforming Agricultural Supply Chains with Salesforce 🚀

---

## 📖 Overview

**AgriEdge Orange Ltd** is revolutionizing the agricultural supply chain by leveraging a **Salesforce-based Order Management System (OMS)**.
This project focuses on digitizing core operations, reducing manual effort, and enabling **data-driven decision-making** for agricultural businesses.

---

## 🚀 Problem Statement

Traditional agricultural order management systems often struggle with:

* ❌ Manual and time-consuming order processing
* ❌ Lack of real-time tracking and visibility
* ❌ Inefficient inventory management
* ❌ Poor communication between farmers, suppliers, and customers

---

## 💡 Proposed Solution

Our platform introduces a **Salesforce-powered digital ecosystem** that:

* ✅ Automates end-to-end order processing
* 📡 Provides real-time tracking of orders
* 🤝 Enhances customer and farmer relationship management
* 📦 Improves inventory visibility and supply chain coordination

---

## 🛠️ Tech Stack

| Layer          | Technology Used                                       |
| -------------- | ----------------------------------------------------- |
| 💻 Frontend    | Lightning Web Components (LWC), HTML, CSS, JavaScript |
| ⚙️ Backend     | Apex (Classes & Triggers)                             |
| 🗄️ Database   | Salesforce Objects                                    |
| 🔍 Querying    | SOQL (Salesforce Object Query Language)               |
| 🔗 Integration | REST APIs for external systems                        |

## 🔄 Workflow Overview

```plaintext id="flow982"
1. User (Farmer/Customer) logs into the system
        ↓
2. Creates a new order via LWC interface
        ↓
3. Apex Controllers validate and process the request
        ↓
4. Order details stored in Salesforce Objects
        ↓
5. Triggers automate updates (inventory, status)
        ↓
6. Notifications sent to stakeholders
        ↓
7. Order status tracked in real-time
        ↓
8. Data reflected in Reports & Dashboards
```
